## Approach \#1: `verbatimEnvironments`

You can render code with custom components:

```ts sveltex.config.ts
export const sveltex = await sveltexPreprocessor({
    general: {
        verbatimEnvironments: {
            Code: {
                processInner: 'code',
                forwardAllAttributes: true,
            },
        },
    },
    markdown: {
        code: {
            backend: 'highlight.js',
        },
    },
});
```

Now, consider the following snippet:

```md
<Code lang='ts'>
const example = new Map<string, { prop: number[] }>();
</Code>
```

This snippet would be transformed into the following:

```md
<Code lang='ts'>
<span class="hljs-keyword">const</span> a = <span class="hljs-keyword">new</span> <span class="hljs-title class_">Map</span>&lt;<span class="hljs-built_in">string</span>, &lbrace; <span class="hljs-attr">prop</span>: <span class="hljs-built_in">number</span>[] &rbrace;&gt;();
</Code>
```
